Problem Statement

Improper waste segregation leads to environmental pollution, inefficient recycling, and increased landfill usage. Manual segregation is unreliable and not scalable.

This project presents an AI-driven smart waste management system that automatically classifies waste images, simulates IoT-based bin operations, and maintains transparent records using blockchain-inspired principles.

System Workflow

  • User uploads a waste image
  • AI model predicts ImageNet labels
  • Semantic mapper converts labels to waste categories
  • Decision engine validates or rejects the waste
  • IoT-based bin simulation opens the correct bin
  • Action is logged in blockchain-style ledger

System Architecture

Waste Categories

  • Wet Waste: Food waste, fruit peels
  • Dry Waste: Paper, cardboard
  • Recyclable: Plastic bottles, glass, cans
  • E-Waste: Chargers, laptops, keyboards
  • Unclassified: Mixed or unclear waste

Technologies

Results

The system accurately classifies waste, rejects mixed inputs, simulates smart bins, and logs actions securely. It demonstrates a scalable architecture suitable for smart city waste management solutions.
